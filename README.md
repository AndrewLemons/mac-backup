# Mac Backup

Tools to selectively backup a Mac.

## Backup

Backup to the provided location.

```sh
./src/backup.sh <backup_dir>
```

## Restore

Restore a backup from the provided location.

```sh
./src/restore.sh <backup_dir>
```

## Supported

- Firefox
- Orca Slicer
- Prism Launcher
- User Developer folder
- User Documents folder
